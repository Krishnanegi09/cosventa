# Cosmetic Store Management System

## Description
This is a Python-based desktop application developed using Tkinter and MySQL.
It helps manage cosmetic products, customers, and billing operations.

## Features
- Admin login
- Product management
- Customer login
- Billing system
- Automatic stock update
- Sales record storage

## Technologies Used
- Python
- Tkinter
- MySQL

## How to Run
1. Import database.sql into MySQL
2. Update database credentials in db.py
3. Run:
   python3 main.py
